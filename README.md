# AI Marketing Automation

Набор скриптов для автоматизации рутинных задач в маркетинге и закупке трафика (User Acquisition). Проект решает задачи массовой загрузки рекламы, автоматической отчетности и оптимизации кампаний с помощью ИИ.

## Состав проекта

### 1. Автоматизация загрузки креативов
* **Скрипты:** `Meta creative only upload`, `app creative upload Meta`, `upload web creatives Meta`, `Unity upload creatives`.
* **Что делает:** Массово выгружает видео и статику в рекламные кабинеты Facebook (Meta) и Unity Ads.
* **Логика:** Ручная заливка десятков креативов под разные гео и форматы отнимает часы и ведет к человеческим ошибкам. Работа напрямую через API рекламных сетей гарантирует точность и экономит время медиабайера.

### 2. Аналитика и отчетность
* **Скрипты:** `Daily report script`, `creative report auto upload`, `predict calculation`.
* **Что делает:** Сводит ежедневную статистику, обновляет данные по креативам и рассчитывает предикты (окупаемость, ROAS).


### 3. AI Анализ и Оптимизация
* **Скрипты:** `UA AI optimization`, `applovin geo ai analysis`.
* **Что делает:** Использует ИИ для корректировки ставок/бюджетов и поиска неочевидных связок по различным гео.
* **Логика:** Алгоритмы могут анализировать массив данных быстрее человека. ИИ нужен для поиска скрытых микротрендов на отдельных гео, которые невозможно заметить при ручном переборе в Excel.
